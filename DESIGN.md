# Gravix Omni-Hub Design System

This file serves as the single source of truth for the Hub's aesthetic. The `uiMatrix` dynamically parses these variables to inject CSS directly into the application.

## Color Palette (60-30-10 Rule)
- **Background (60%)**: `#0a0a0a`
- **Secondary (30%)**: `#141414`
- **Tertiary (Surface)**: `#1f1f1f`
- **Accent (10%)**: `#3b82f6`

## Semantic Colors
- **Success**: `#10b981`
- **Warning**: `#f59e0b`
- **Error**: `#ef4444`
- **Info**: `#3b82f6`

## Typography
- **Primary Font**: `Space Grotesk`
- **Secondary Font**: `Inter`

## Constraints & Layout
- **Border Radius (Cards)**: `12px`
- **Border Radius (Buttons)**: `6px`
- **Border Thickness**: `1px`
- **Glass Opacity**: `0.85`

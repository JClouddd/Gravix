# Secret Manager for Gravix
## Create Secret: gcloud secrets create NAME --data-file=- (pipe value)
## Access Secret: gcloud secrets versions access latest --secret=NAME
## Rotation: Set rotation_period and next_rotation_time on secret metadata
## IAM: Grant roles/secretmanager.secretAccessor to service account
## Versions: Each update creates a new version. Latest alias points to newest.
## Automatic Replication: Default is automatic. Manual for specific regions.
## Secret Labels: metadata.labels for organization (env, service, team)
## Gravix Secrets: gemini-api-key, jules-api-key, google-oauth-client-secret
